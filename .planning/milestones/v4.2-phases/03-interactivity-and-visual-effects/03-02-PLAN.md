  <action>
1. **CustomCursor.tsx** -- Create client component:
   - `"use client"` directive
   - Render a fixed-position div (the dot): 12px circle, `var(--text-primary)` background, `pointer-events-none`, `z-[9999]`, `transform: translate(-50%, -50%)`, `hidden md:block`
   - Use `gsap.quickTo()` for smooth cursor following (duration: 0.3, ease: "power3.out") on x and y
   - Add mousemove listener on window to feed coordinates to quickTo
   - **Touch detection:** Check `'ontouchstart' in window || navigator.maxTouchPoints > 0` on mount. If touch device, return null (render nothing) and attach no listeners.
   - **Hover state:** Query all elements with `[data-magnetic]` attribute. On mouseenter: animate cursor dot to larger size (32px) and accent color (`var(--accent-purple)` / #a78bcd). On mouseleave: animate back to 12px and primary color.
   - **Magnetic pull:** On mousemove, for each `[data-magnetic]` element, calculate distance from cursor to element center. If within 50px radius, apply `gsap.to(element, { x: distX * pull * 0.3, y: distY * pull * 0.3, duration: 0.3, ease: "power2.out", overwrite: true })`. On mouseleave, reset with `gsap.to(element, { x: 0, y: 0, duration: 0.5, ease: "elastic.out(1, 0.3)" })`.
   - **Cleanup:** Remove all event listeners in useEffect return. Use useGSAP for GSAP-specific cleanup.

2. **layout.tsx** -- Add CustomCursor:
   - Import CustomCursor
   - Render `<CustomCursor />` inside body, outside providers (at root level so it's always visible): `<body><CustomCursor /><DarkModeProvider><SmoothScrollProvider>{children}</SmoothScrollProvider></DarkModeProvider></body>`

3. **Add data-magnetic attributes** to interactive elements:
   - NeoBrutalButton: add `data-magnetic` to the rendered `<a>` or `<button>` element
   - FloatingNav dots: add `data-magnetic` to each nav button
   - Contact section links (in Exploration6.tsx): ensure they have `data-magnetic`
