**MobileNav.tsx** -- Create new component following FloatingNav patterns:

Structure:
- `"use client"` directive
- Import `useState`, `useRef`, `useCallback` from React
- Import `gsap` from `@/lib/gsap` (same import path as other components)
- Import `useLenis` from `lenis/react`

Sections array (same as FloatingNav):
```typescript
const sections = [
  { id: "about", label: "About" },
  { id: "experience", label: "Experience" },
  { id: "skills", label: "Skills" },
  { id: "projects", label: "Projects" },
  { id: "contact", label: "Contact" },
];
```

State: `isOpen` boolean, `drawerRef` for the overlay div, `linksRef` for stagger targets.

Hamburger button:
- Fixed top-4 right-4, z-[60]
- `md:hidden` (only visible below 768px)
- 44px circle with glass styling: `background: var(--glass-fill)`, `backdropFilter: blur(12px)`, `border: 2px solid var(--glass-border)`, `boxShadow: 2px 2px 0px var(--glass-border)`
- Renders hamburger icon (3 lines) when closed, X when open -- use simple span elements with CSS transforms, not an icon library
- `aria-label` toggles between "Open menu" and "Close menu"
- BEM class: `mobile-nav__hamburger`

Drawer overlay:
- Keep mounted but use `visibility: hidden` + `opacity: 0` when closed (avoids GSAP animating unmounted elements -- Pitfall 2 from research)
- Fixed inset-0, z-[55]
- Full-screen frosted glass: `background: linear-gradient(160deg, var(--glass-fill) 0%, var(--glass-fill-end, var(--glass-fill)) 100%)`, `backdropFilter: blur(40px) saturate(1.6)`
- Flex column centered

Open animation (GSAP timeline):
- Set drawer visible
- `lenis?.stop()` to lock scroll
- Fade in + scale: `fromTo` opacity 0->1, scale 0.95->1, duration 0.4, ease "power3.out"
- Stagger links: `fromTo` `.mobile-nav__link` elements, y 20->0 opacity 0->1, stagger 0.06, duration 0.3, ease "power3.out", overlap "-=0.2"

Close animation:
- `gsap.to` drawer: opacity 0, scale 0.95, duration 0.3, ease "power2.in"
- `onComplete`: set visibility hidden, `setIsOpen(false)`, `lenis?.start()`

ScrollTo handler:
- Call `closeDrawer()`
- After 350ms timeout (let close animation finish), call `lenis?.scrollTo(`#${id}`, { offset: -80, duration: 1.2 })`

Nav links in drawer:
- Centered vertically in a flex column with gap-6
- Each link: `mobile-nav__link` class, text-2xl font-bold, `color: var(--text-primary)`, `font-family: var(--font-display)`
- Button elements with `onClick={() => scrollTo(section.id)}`
- Min-height 44px for touch targets (RESP-04 compliance)

Dark mode toggle:
- Render `<DarkModeToggle size={44} />` below the nav links with `mt-8`

Click-outside to close:
- Add `onClick` on the overlay div itself that calls `closeDrawer()`
- Use `e.stopPropagation()` on the inner nav container to prevent closing when tapping links area
